# Contributing

## Writing a post

1. Copy `posts/template.md` to `posts/incoming/YYYYMMDD.NNN.md`
2. Fill in the frontmatter
3. Write each section in your own voice
4. When ready, move to `posts/` and update `index.html`

## Voice guidelines

- No em dashes
- Declarative sentences
- First person, present tense where possible
- End with Make it better — open the door for the reader
