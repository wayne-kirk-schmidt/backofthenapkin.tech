# Design Reference

## Colors

| Variable | Value | Use |
|----------|-------|-----|
| --ink | #0f2742 | Dark background base |
| --sky | #7eb3f5 | Accent, highlights |
| --accent | #3b6fd4 | Links, active states |
| --brand | #244a86 | Headers, kicker text on light pages |

## Typography

| Use | Font | Weight |
|-----|------|--------|
| All body | DM Sans | 300, 400, 500, 700, 800 |
| Metadata, labels, nav | IBM Plex Mono | 400, 500 |

## Pages

| File | Background | Use |
|------|------------|-----|
| index.html | background.png (dark) | Homepage, post list |
| post.html | #f5f2ec (light) | Article reading surface |
